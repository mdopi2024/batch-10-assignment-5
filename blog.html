What is event delegation in the content of DOM, and why is it used

Event delegation is a technique in JavaScript where a single event listener is attached to a parent element, it is handle events triggered by its child elements. it is very helpfull.If new elements are added to your document that require event handling, you don't need to attach a new event handler each time.

How do you manipulate an elements atributes and styles using the DOM

DOM manipulation allows us to interact with and modify HTML elements dynamically. we can enhance user interaction, change page content, and update styles in real-time.
