What is the Document Object Model(DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a web page as a tree of objects, allowing programming languages like JavaScript to access and manipulate HTML and CSS. The DOM treats an HTML or XML document as a hierarchy of nodes (such as elements, attributes, and text), enabling dynamic interaction with the content.

What is event delegation in the content of DOM, and why is it used

Event delegation is a technique in JavaScript where a single event listener is attached to a parent element, allowing it to handle events triggered by its child elements. Instead of attaching event listeners to each child element individually, you delegate the responsibility to a common ancestor (usually a parent or container element). This is made possible by event bubbling, which is the process where an event triggered on a child element first executes its own event listener and then propagates up to its parent elements in the DOM tree, eventually reaching the document's root unless stopped.
